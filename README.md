1) What is the difference between var, let, and const?
 -> var is old and function-scoped, let is block-scoped and changeable, const is block-scoped and cannot be reassigned.

2) What is the difference between map(), forEach(), and filter()?
-> map makes a new array, forEach just loops without returning, filter makes a new array with only items that pass a condition.
